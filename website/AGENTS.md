# Rspress 文档站维护说明

本目录是 Ocean 课程设计项目的 Rspress 补课站，面向零基础或基础薄弱的组员。

## 命令

- `pnpm run dev`：本地启动文档站
- `pnpm run build`：生产构建
- `pnpm run preview`：本地预览构建结果

## 写作原则

- 主语言是简体中文，不需要多语言。
- 内容要围绕本项目真实代码，不写泛泛的 PHP 教程。
- 默认读者基础薄弱，要用大白话解释术语。
- 答辩技巧是次要用途；第一用途是补习项目用到的 PHP、数据库和 Laravel 知识。
- 站点内容按学习主题组织：共同基础、Laravel 后端、数据库、按分工学习、答辩问答。
- 每页尽量包含：这页解决什么问题、要看哪些项目文件、用大白话解释、必背说法、常见追问。
