# 海洋巡检课程设计前端

React + TypeScript + Vite 实现的简化后台界面，用于课程设计和答辩演示。

## 本地运行

```bash
pnpm install
pnpm run dev
```

默认接口地址为 `http://127.0.0.1:8000/api`。如需修改，可创建 `.env.local`：

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000/api
```

## 页面

- 首页统计
- 巡检任务
- 样本登记
- 检测结果
- 异常与分析
- 答辩说明

## 设计原则

- 使用海洋/青绿色后台风格。
- 保持“侧边栏 + 顶部标题 + 卡片/表格/表单”的简单结构。
- 不引入复杂状态库，便于本科课程答辩讲解。
- 后端未启动时显示演示数据，方便单独预览前端。
