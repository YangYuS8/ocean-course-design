# 海洋环境巡检与综合分析平台前端

React + TypeScript + Vite 实现的轻量后台管理界面，用于巡检任务、样本、检测结果、异常处置与分析建议的闭环管理。

## 本地运行

```bash
pnpm install
pnpm run dev
```

默认接口地址为 `http://127.0.0.1:8000/api`。如需修改，可创建 `.env.local`：

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000/api
```

## 页面

- 首页统计
- 巡检任务
- 样本登记
- 检测结果
- 异常与分析
- 系统说明

## 设计原则

- 使用海洋/青绿色后台风格。
- 保持“侧边栏 + 顶部标题 + 卡片/表格/表单”的简单结构。
- 不引入复杂状态库，保持页面逻辑清晰可维护。
- 后端未启动时使用本地离线数据，方便独立预览前端。
